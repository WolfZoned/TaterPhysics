# TaterPhysics
A 2D impulse based physics engine for Java.

# Current Features
Rendering\
Basic interactions
Bounding box

# Needs Porting
GJK, EPA, and impulse calculations

Friction

Circles

A Customizable Joint System Allowing For:\
--Fixed joints\
--Pivot joints\
--Wheel motor joints\
--Stepped motor joints\
--Rope joints\
--Piston joints\
--And more

Multiple interaction modes\
--Drawing shapes with custom properties\
--Creating custom joints between 2 objects\
--Moving shapes\
--Adding points to use for joints
